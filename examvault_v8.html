    STRICT AI EVALUATION MODE — ExamVault Booklet I am uploading my ExamVault AI Evaluation Booklet PDF. This PDF contains: 1. Exam metadata (test name, candidate, date, duration, marking scheme) 2. An encoded answer key with my responses, question types, and time per question 3. Individual question images (one per page) 4. (Optional) An answer sheet image/PDF uploaded by the student STRICT RULES — YOU MUST FOLLOW THESE WITHOUT EXCEPTION: - DO NOT guess or hallucinate correct answers. If you cannot determine the correct answer from the question image, say "CANNOT DETERMINE" — never make up an answer. - DO NOT give partial credit where the marking scheme says 0 or negative. - APPLY the exact marking scheme stated in the metadata: +M for correct, -M for wrong. No exceptions. - For Integer/Decimal questions: answer must match EXACTLY (no rounding unless explicitly allowed). - For Multiple Correct: apply partial marking strictly as encoded (partial correct = proportional, any wrong = negative). - For Scan answers: compare against the answer sheet if uploaded. If no answer sheet, mark as "PENDING MANUAL CHECK". - If the question image is unclear, say "IMAGE UNCLEAR — MANUAL CHECK REQUIRED". TASKS: A) READ metadata and answer key from first pages carefully. B) EVALUATE each question image. For each: - Question number and type (MCQ/Multi/Integer/Scan) - My answer (from answer key) - Correct answer (from question image) - Marks: STRICTLY +M, -M, or 0 per the scheme - One-line solution C) MARKSHEET: - Total score (sum of all marks, positives and negatives) - Correct / Wrong / Skipped / Pending counts - Section-wise breakdown if identifiable - Time analysis (slowest 5 questions) D) DETAILED SOLUTIONS for every WRONG answer (step-by-step). E) If I ask "explain Q[N]" — give full working for that question only. START: Confirm you can see metadata and answer key, then proceed.
